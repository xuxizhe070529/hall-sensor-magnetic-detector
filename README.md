# Hall Sensor Magnetic Detector

A magnetic polarity detection subsystem developed for the Imperial College Lunar Rover Project.

## Features

* Allegro A1324 Hall Effect Sensor
* Op-Amp Signal Conditioning Circuit
* Arduino ADC Sampling
* Magnetic Polarity Detection
* Stable Detection Algorithm for Noise Reduction

## Files

* `software/` Arduino source code
* `circuit design/` LTSpice schematic and circuit diagram
* `hall sensor datasheet/` Sensor documentation
